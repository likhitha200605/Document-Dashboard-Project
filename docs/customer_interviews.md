# Customer Interviews Documentation

## Interview 1 – NGO Representative

Name: Mr. Ramesh Kumar
Role: NGO Program Coordinator

Q1: What type of reports do you usually prepare?
A: Monthly beneficiary reports, donation utilization reports, and impact analysis reports.

Q2: What tools do you currently use to create reports?
A: Excel spreadsheets and manual documentation.

Q3: How much time does it take to prepare a report?
A: Around 3–4 hours.

Q4: What difficulties do you face while preparing reports?
A: Consolidating data and creating visual summaries.

Q5: How often do you generate reports?
A: Weekly and monthly.

Q6: Would an AI system that generates reports automatically help you?
A: Yes, it would save time and improve efficiency.
